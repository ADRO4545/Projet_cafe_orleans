# Café Orléans (Coda Bakery)

## Description
Ce projet est le code source d'un site vitrine "one-page" pour un café-boulangerie fictif nommé "Coda Bakery". Le site est conçu pour être moderne, attrayant et entièrement responsive.

## Aperçu
Le projet consiste en une page HTML unique (`index.html`) et une feuille de style (`styles.css`) qui gère l'ensemble de la mise en page et de l'apparence. Il comprend également du JavaScript embarqué pour gérer l'interactivité, comme le menu de navigation mobile et un bouton "Retour en haut".

## Fonctionnalités
- Design moderne et responsive
- Menu de navigation mobile
- Bouton "Retour en haut" pour une navigation fluide
- Sections dédiées pour présenter les produits, les services et les informations de contact

## Technologies Utilisées
- HTML5
- CSS3
- JavaScript

## Installation
Pour utiliser ce projet localement, suivez ces étapes :

1. Clonez ce dépôt sur votre machine locale.
2. Ouvrez le fichier `index.html` dans votre navigateur préféré.

## Utilisation
Ouvrez simplement le fichier `index.html` dans un navigateur web pour visualiser le site.
